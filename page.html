Tugas Besar Jaringan Komputer!

Tugas besar ini berkaitan dengan pembuatan web server berbasis TCP yang mampu menangani permintaan HTTP menggunakan socket programming. Server harus dapat menerima dan memproses request, mengirimkan file yang diminta, dan jika file tidak ditemukan, server akan mengirimkan response "404 Not Found". Selain itu, server harus mampu menangani beberapa permintaan secara bersamaan dengan memanfaatkan multithreading, di mana setiap permintaan dilayani dalam thread terpisah. Selain server, tugas ini juga mencakup pembuatan klien HTTP yang dapat mengirimkan request dan menampilkan response dari server.

Kesimpulan

Proyek web server berbasis TCP ini berhasil mengimplementasikan konsep fundamental jaringan komputer. Melalui penggunaan socket programming, kami berhasil membuat server yang dapat menangani permintaan HTTP secara efisien dengan dukungan multithreading.

Pencapaian Utama:

  • Implementasi TCP socket untuk komunikasi client-server
  • Pengolahan request HTTP dengan parsing yang tepat
  • Sistem response yang sesuai dengan standar HTTP
  • Penanganan error 404 untuk file yang tidak ditemukan
  • Multithreading untuk menangani koneksi simultan

Pembelajaran:

Proyek ini memberikan pemahaman mendalam tentang arsitektur jaringan, protokol TCP/IP, dan bagaimana web server bekerja di level fundamental. Implementasi multithreading juga memberikan wawasan tentang pentingnya concurrent programming dalam sistem jaringan modern.
